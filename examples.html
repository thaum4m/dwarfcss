Dwarf CSS

Dwarf CSS is a very light-weight CSS framework which consists of styles grouped into standalone files which can be used individually in your own projects.

1. Grid

The following is a 12-column responsive grid.

It uses a subset of Bootstrap's grid classes prefixed with "d-". In addition, the classes "d-row-*-flex" are used to give columns equal height.

1.5. Grid classes

Class Description
d-row Enables grid column styles and provides 6px spacing between columns as well as a clearfix.
d-row-compact Enables grid column styles without spacing between columns as well as applying a clearfix.
d-col-xs-x Defines a floated column of x units width independent of the browser viewport width.
d-col-sm-x Defines a floated column of x units width only if the browser viewport width is above 767px.
d-col-md-x Defines a floated column of x units width only if the browser viewport width is above 991px.
d-col-lg-x Defines a floated column of x units width only if the browser viewport width is above 1199px.
